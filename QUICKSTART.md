# Quick Start Guide for Clean Web Proxy Project

## Introduction
This guide provides a quick overview to help you get started with the Clean Web Proxy project.

## Prerequisites
- Ensure you have Python 3.6 or higher installed.
- Install necessary dependencies.

```bash
pip install -r requirements.txt
```

## Running the Proxy
You can start the web proxy server by running the following command:

```bash
python app.py
```

## Accessing the Proxy
After starting the server, you can access the web proxy at `http://localhost:8080`.
